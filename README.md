# ChatGPT plugin using Node.js

ChatGPT plugin developed with Node.js

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod

# api-plugin.json
http://localhost:3000/.well-known/api-plugin.json

# OpenAPI
http://localhost:3000/api-yaml
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
